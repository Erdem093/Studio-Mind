# 5-10 Minute Demo Runbook

## 0. Setup (30s)

- Open deployed app landing page.
- Mention this is built for Anyway + Animoca tracks.
- Log in and show required onboarding gate.

## 0.5 Onboarding (60s)

- Complete channel goal, style settings, and inspiration links.
- Generate baseline prompt and confirm save.
- Mention this baseline is injected into every future run.

## 1. Problem (45s)

- "Creators juggle scripting, hooks, titles, and strategy manually."
- "This platform turns one idea into a coordinated AI content pipeline."

## 2. Product Flow (2m)

- Go to dashboard, open a video project.
- Click `New Run`.
- Show generated outputs in run detail:
  - hook
  - script
  - title
  - strategy

Highlight that each artifact is produced by a specialized agent.

## 3. Memory + Cognition (1.5m)

- On run detail, submit artifact-targeted feedback (e.g. ScriptAgent -> `too_long` + custom note).
- Trigger another run and call out:
  - shorter script structure
  - `memory_applied` references visible in observability
  - updated quality score (`quality_delta`)

## 4. Anyway Observability (1.5m)

- Open observability page.
- Show run trace link and per-agent metrics:
  - tokens
  - latency
  - cost
  - failures (if any)
  - collector export status

Narrative: "Every agent step is measurable and production-observable."

## 5. Commercialization (1.5m)

- Open billing page.
- Show plan/subscription state.
- Show Connect section:
  - `Connect payouts (demo)`
  - `Pay with platform fee (demo)`

Narrative: "The same system supports SaaS subscriptions and marketplace-style fees."

## 6. Closing (45s)

- Anyway angle: end-to-end traceability for AI pipelines.
- Animoca angle: persistent multi-agent content team with memory.
- Mention roadmap: richer memory, additional channels, deeper partner integrations.
