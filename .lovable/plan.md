

# ContentPilot — AI Content Creation Platform

## Overview
An AI-powered content creation system for challenge-based video creators (YouTube, Instagram). The app acts as a "content autopilot" — helping creators plan, script, optimize, and analyze videos through iterative AI pipeline runs with human-in-the-loop approvals.

---

## Pages & Features

### 1. Authentication
- Email/password signup and login using Supabase Auth
- User profile with creator name and avatar
- Protected routes — all pages require login

### 2. Dashboard
- **Video list** showing all the user's content projects (title, status, latest run summary, date)
- Quick stats at the top: total videos, runs this month, runs remaining on plan
- Button to create a new video project
- Each video card links to its detail/run history

### 3. Video Detail & Runs
- Video overview: idea, status, created date
- **Run history** — list of all AI pipeline runs for this video, newest first
- Each run shows: timestamp, status (running/completed/failed), number of artifacts generated
- Button to trigger a new run (UI only for now — no actual AI pipeline)
- Click a run to open the **Run Detail View**

### 4. Run Detail View
- Shows all **artifacts** generated by that run, grouped by type:
  - Story structure / outline
  - Script drafts
  - Hook options
  - Title & thumbnail ideas
- Each artifact has:
  - Content preview
  - **Approve / Reject** buttons (human-in-the-loop)
  - Status badge (pending / approved / rejected)
- Ability to compare artifacts across runs (side-by-side view)

### 5. Billing Page
- Current plan display (Starter / Pro) with usage stats (runs used / runs included)
- Stripe integration for subscription management
  - Connect payment method via Stripe Checkout
  - Sandbox/test mode for demo
- Upgrade/downgrade between plans
- Usage-based overage display

### 6. Observability View
- List of recent runs with cost breakdown per run (tokens/API calls consumed)
- Cost summary: total spend this billing period
- Trace link placeholder for each run (link to external trace/log viewer)
- Simple chart showing usage over time

---

## Database Schema (Supabase)

### Core Tables
- **profiles** — user profile linked to auth.users (name, avatar, created_at)
- **videos** — content projects (user_id, title, idea/description, status, created_at)
- **runs** — AI pipeline executions (video_id, status, cost_tokens, cost_usd, started_at, completed_at)
- **artifacts** — generated outputs (run_id, type [script/hook/title/story], content, approval_status [pending/approved/rejected], approved_at)

### Security
- Row-Level Security on all tables scoped to the authenticated user
- Users can only see and manage their own videos, runs, and artifacts
